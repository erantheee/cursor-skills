# 丝印识别与三轮验证参考

## 三轮验证流程

对每颗**关键 IC** 强制执行，不得跳过：

```
Round1 读丝印   → 只记录原文/Logo/磨标/屏蔽罩镭雕，不猜型号
Round2 查标识   → WebSearch "{丝印}" marking OR datasheet OR teardown
Round3 电路印证 → 封装 + 邻件 + 品类应用；排除 2 个兄弟型号
```

Round3 未通过 → 置信度最高 **中**；仅 Round1 → **低**。

### Round2 检索模板

```
"{丝印原文}" chip marking
"{丝印原文}" datasheet OR 规格书
"{推测型号}" package {封装类型}
site:52audio.com OR site:ifixit.com "{产品名}" 拆解
```

## 屏蔽罩模组（金属罩 LTE/BT 模组）

**规则**：罩面镭雕/印刷标识 = 主识别依据，通常可达 **高** 置信度。

| 厂商 | 罩面常见标识 | 注意 |
|------|--------------|------|
| Quectel 移远 | `EC600K-CN`、`EG25` 等 + IMEI + CCC | 罩内 ASR/高通芯片不可见，**不猜罩内** |
| Telit | 型号 + IMEI | 同上 |
| Simcom | 型号 + 认证标 | 同上 |
| u-blox | GNSS 模组型号 | 非蜂窝 |

拍照要求：罩面**全部镭雕**清晰（型号、SN、IMEI、认证标）。反光时用侧光，见 reference.md 拍照指南。

## 常见丝印快速解码

### NOR Flash（8pin SOIC / WSON）

| 丝印片段 | 推测型号 | 厂商 |
|----------|----------|------|
| `25Q32` / `25Q32JV` | W25Q32（4MB） | Winbond |
| `25Q64` | W25Q64（8MB） | Winbond |
| `GD25Q32` | GD25Q32 | GigaDevice |
| `P25Q32` | P25Q32 | Puya |

容量：`25Q**` 中 Q 后数字 ≈ Mbit（32→4MB）。

### 蓝牙音频 SoC

| 丝印片段 | 厂商/系列 | 备注 |
|----------|-----------|------|
| `JL` + `AC69` | 杰理 AC696x 系 | 入门 TWS 常见；非中科蓝讯（无 BLU 标） |
| `JL` + `JL7016/7018/7006` | 杰理 JL701N 系 | 中高端 TWS/OWS；批次码如 `BP2X…` **非型号** |
| `BES` / `B` 标 + `2700/2600/2500` | 恒玄 BES | 常 QFN/BGA；与杰理 **不可** 仅凭封装互换 |
| `BLU` / `BT89xx` | 中科蓝讯 | 非杰理 |
| `AB15xx` / Airoha 标 | 络达 | 封装与 JL 不同需核对 |
| `BT892x` / 物奇标 | 物奇/其他 | 低清时列候选，勿写死 |

**兄弟型号排除（音频 SoC）**

| 看到 | 不要混淆为 | 区分依据 |
|------|------------|----------|
| 杰理 `JL` + `JL7018` | 恒玄 BES | 顶标字符：`JLxxxx` vs `BESxxxx` |
| 杰理 `JL` + `JL7018` | 杰理 AC696 | 同厂不同代，看第二行前缀 |
| 杰理 / 恒玄 音频 SoC | STM32 / GD32 | 有 `JL`/`BES` 时 **不** 进入通用 MCU 对标流程 |
| 仅 `JL` Logo 无型号行 | 恒玄 BES | **禁止** 凭价位/品类偏见定稿 BES |

### Logo-only 铁律（音频 SoC 及所有关键 IC）

- **禁止** 仅凭 Logo 或图像描述将厂商定为「高/中」置信度
- Logo + 封装 → 最高 **低**；须见 **第二行型号**（如 `JL7018F3`）才可升到 **中/高**
- 报告须写 **丝印原文**；若只有 Logo，输出：「待微距确认型号行」
- **禁止** 把图像描述中的厂商名当已验证事实（须与用户可见丝印或 Round2 检索一致）
- **价位 ≠ 厂商**：高端 4G 耳机 ≠ 必然恒玄；杰理 JL7018 已是中高端档

### 蜂窝模组兄弟型号排除

| 看到 | 不要混淆为 | 区分依据 |
|------|------------|----------|
| EC600**K**-CN | EC600M / EC600E | 罩面型号字母；K=Cat1 bis 新系列 |
| EC600M | EC600K | 区域后缀与料号不同 |
| EG25-G | EC600 系列 | EG25 为更高阶 4G 模组，体积更大 |

### 电源 / 充电 IC

| 丝印片段 | 推测 | 置信度前提 |
|----------|------|------------|
| `TP4056` | 线性充电 | 丝印完整可见 |
| `SY88` 开头 | 硅力杰充电/PMIC 系 | 需第二行确认，否则标候选 |
| 丝印磨损 SOT23-6 | 列 2 个候选 | 不得超过「低」 |

**禁止**：丝印磨损时写死 SY8801 等具体型号。

## 日期码解读

常见 4 位周码 `YYWW` 或 `WWYY`（依厂商而异）：

| 丝印 | 解读 | 置信度 |
|------|------|--------|
| `2349` | 常见为 2023 年第 49 周 | 中（需结合厂商规则） |
| `2445` | 2024 年第 45 周 | 中 |

仅用于判断生产批次新旧，**不能**单独用于芯片型号判定。

## 兄弟型号排除（Round3 必做）

对每颗 **中** 及以上置信度的 IC，列出 2 个最像兄弟型号并排除：

```markdown
### {推测型号}
- 兄弟候选 A：{型号} — 排除：{封装脚位不同 / 品类不符 / 周边电感数量不对}
- 兄弟候选 B：{型号} — 排除：{…}
```

### 常用排除维度

1. **封装**：QFN32 4×4 vs QFN48 5×5 不可互换
2. **邻件**：4G 模组旁有大电流电感；BT SoC 旁有 32.768kHz 晶振
3. **品类**：充电宝 PMIC 不应出现在 TWS 耳机的同位置
4. **射频**：弹簧天线多接蜂窝模组；PCB 倒 F 多接 BT
